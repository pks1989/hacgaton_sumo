Хакатон "Цифровой прорыв". Команда solution-now

Система оптимизации городского траффика по заданным параметрам (среднее время пути и средняя скорость).

В рамках хакатона, для небольшой части города (перекрёсток 9 мая - Авиаторов и окрестности). С помощью библиотека Eclipse SUMO 
создана модель дорожной сети и траффика.

Написан скрипт который принимает значимые параметры (интервалы всех светофорныых секций на данном перекрёстке), запускает симуляцию
(параметры траффика машин/ч ориентировочные и полученны от экспертов) и по итогам симуляци выдаёт среднее время пути и средниюю скорость
траффика при таких интервалах.

Используя этот скрипт, мы с помощью эволюционного алгоритма (используется Deap Framefork) ищем оптимальные значения интервалов светофоров,
при которых вермя будет минимальным, но при этом скорость движения траспорта не станет опасной.
